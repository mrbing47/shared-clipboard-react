## Realtime Shared Clipboard

A clipboard with realtime syncing of information like links, text snippets (or whatever you want) across all your devices.

## Tech Stack

1. Firebase Auth
2. Firebase Firestore
3. React

## How to run?

1. `npm i`
2. `npm start`
